# Agent Max 修改说明

本版目标：站点定位就是代理转化，SEO 承接搜索词，页面负责把人导向 ZEOGPT。

## 比上一版更狠的地方

1. 全站顶部新增绿色转化条：ZEOGPT 注册入口。
2. 全站新增滚动/停留触发的转化弹窗，用户可关闭。
3. 所有 CTA 从“说明型”升级为“立即打开 ZEOGPT”。
4. 文章页、专题页、博客页、FAQ、套餐页、ZEOGPT 页结尾再加一次 CTA。
5. 首页主按钮继续强化为“立即打开 ZEOGPT”。
6. 首页路径文案改为代理转化路径。
7. 后续批量导入精修文章时，自动补文章顶部和底部 CTA。
8. 视觉上增加“推荐入口”角标、强按钮、顶部条、弹窗和右下角悬浮入口。

## 仍然保留的底线

1. 不冒充 OpenAI 官方。
2. 不写“官方中文版”“永久稳定”“100%可用”“绝不封号”。
3. 外链使用 sponsored / nofollow。
4. 仍提示价格、额度和规则以 ZEOGPT 实时页面为准。
