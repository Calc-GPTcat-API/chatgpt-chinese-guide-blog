# 关于 ChatGPT中文指南

**ChatGPT中文指南** 是一个独立中文教程站，帮助读者核对 ChatGPT 官方入口、处理常见使用问题，并理解官方服务、镜像站、代充、共享账号和第三方多模型平台之间的区别。

本站不属于 OpenAI，也不代表 OpenAI、Microsoft、Anthropic、Google、xAI、DeepSeek 或 ZEOGPT。涉及产品名称、商标、价格、模型、额度、支持地区和服务规则时，应以对应产品的实时公开页面为准。

## 我们写什么

- 官方入口、账号、登录、中文使用和常见故障排查；
- ChatGPT Plus 订阅、付款、续费、退款与账号安全；
- 镜像站、共享账号、代充和第三方聚合平台的风险核验；
- 经过来源核对、编辑审阅并标注日期的产品说明与比较。

## 编辑与审核

文章统一由“ChatGPT中文指南编辑部”发布。编辑流程包括选题去重、来源核验、正文复核、推广披露、链接检查和构建验收。重要文章会分别显示首次发布、最近更新和信息核验日期。

详细标准见 [编辑与更正政策](/editorial-policy/)。

## 推广与利益关系披露

本站部分 ZEOGPT 链接带有推广参数。读者通过此类链接注册或购买时，本站可能获得推广收益。推广关系不会使 ZEOGPT 成为 OpenAI 官方服务，也不会构成对可用性、稳定性、退款、模型真实性或账号安全的保证。

所有推广链接均应以“第三方服务”“推广链接”等文字明确标识，并使用适当的链接属性。

## 联系与更正

发现事实错误、失效链接、日期过期或披露不清，可通过站点维护者公开的项目联系方式反馈。提交反馈时请附上页面地址、问题位置、建议修改内容和可核验来源。

我们会根据影响范围更新正文、核验日期或更正说明，而不是仅修改页面日期。
