# 关于 ChatGPT中文指南

**ChatGPT中文指南** 是独立的中文 ChatGPT 与 AI 工具教程站，内容重点包括官网入口识别、国内用户使用前核验、Plus 付款注意事项、镜像站安全、ZEOGPT 注册与套餐说明，以及其他 AI 模型工具指南。

## 编辑原则

- 区分官方产品、第三方服务、镜像站、代充与共享账号；
- 对价格、模型、额度、支持地区等易变化信息标注核验日期；
- 优先引用 OpenAI、Microsoft Bing、产品公开页面等一手来源；
- 不使用“已获官方背书”“长期不变”“无条件可用”“账号零风险”等无法证实的承诺；
- 不提供绕过地区限制、规避风控或违反平台规则的方法；
- 文章发生重大变化时更新正文与更新时间，而不是只改日期。
