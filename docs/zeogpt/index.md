# ZEOGPT 是什么？第三方 AI 模型订阅与镜像服务说明

ZEOGPT 的公开首页将产品定位为“AI 模型一站式订阅”，同时提供 ChatGPT 镜像入口与 Codex 相关额度。它把多个模型放在同一服务里，主要面向希望减少多平台切换、使用中文界面或采用国内常见支付方式的用户。

需要先说清楚：**ZEOGPT 不是 OpenAI 官方网站，也不是由 OpenAI 运营的独立中文版本。** 它属于独立运营的第三方服务。第三方平台能否持续提供某个模型、额度是否变化、服务中断如何处理，都应以平台当时的页面和协议为准。

## 适合哪些用户

更可能考虑这类服务的人，通常希望在一个界面切换多个模型，主要做日常问答、写作、翻译、代码辅助，并且可以接受第三方平台在账号归属、数据处理、模型可用性和售后方面与官方服务不同。

如果任务涉及客户资料、未公开源代码、财务数据、医疗信息或其他高敏感内容，优先选择能提供明确数据协议、企业权限和审计能力的服务。

## 支持哪些 AI 模型

ZEOGPT 首页公开列出 GPT、Gemini、Claude、Grok、DeepSeek 等模型系列。多模型聚合的价值在于：写作、长文本、代码、搜索或中文任务可以分别选择不同模型。

但“支持某个模型”不等于始终提供该模型的全部官方功能。第三方服务可能通过不同接口或产品封装提供能力，模型名称、上下文长度、工具调用、文件上传和联网能力也可能与官方产品不同。
