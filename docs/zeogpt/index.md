# ZEOGPT 第三方服务说明

ZEOGPT 是独立运营的第三方多模型平台，**不是 OpenAI 官方服务，也不是 OpenAI 官方中文版**。本站部分 ZEOGPT 链接带推广参数，符合条件的注册或购买可能给本站带来收益。

## 使用前先核对五件事

| 检查项 | 应核对的内容 |
| --- | --- |
| 平台身份 | 运营主体、域名、服务条款和隐私政策 |
| 模型与功能 | 当前页面实际展示的模型、工具、限制和更新说明 |
| 套餐与额度 | 价格、周期、额度计算、并发、降速或公平使用规则 |
| 付款与退款 | 支付方式、自动续费、退款条件、售后渠道和发票 |
| 数据与账号 | 对话保存、数据处理、账号找回、密码和敏感资料风险 |

## 适合与不适合的情况

第三方聚合平台可能适合希望在一个界面比较多个模型、需要本地付款方式或不想分别管理多个订阅的用户。

它不适合需要 OpenAI 官方账号权益、严格企业合规、明确数据处理协议，或不能接受第三方服务变化风险的场景。
