# 隐私说明

本站是静态内容网站，不提供 ChatGPT 账号登录、代充值、对话存储或银行卡收款功能。

## 访问日志

网站托管商、域名服务商或基础设施提供商可能为安全、性能和故障排查记录常规技术日志，例如 IP 地址、浏览器类型、访问时间、请求路径和错误信息。具体处理方式以相应服务商的隐私政策为准。

## 外部链接

点击 OpenAI、ZEOGPT 或其他外部网站后，后续数据处理由对应网站负责。本站无法控制第三方的账号、支付、Cookie、对话保存、模型调用或退款流程。

## 不要提交敏感信息

请勿通过公开评论、代码仓库问题区或其他公开渠道提交密码、验证码、银行卡信息、身份证件、客户资料、公司内部文件或未公开代码。

## 推广参数

部分 ZEOGPT 链接包含推广参数，用于识别推广来源。此类链接会明确标注为第三方和推广链接。是否注册、付款以及第三方如何处理数据，应由读者根据实时条款独立判断。

## 联系与删除请求

本站目前不提供用户账户或站内投稿数据库，因此通常不存在可由本站删除的个人账户资料。若发现页面意外包含个人信息，请通过项目维护渠道反馈具体页面和位置。
