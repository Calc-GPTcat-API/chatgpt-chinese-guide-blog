# 发布前编辑检查

- [ ] 标题与搜索意图一致
- [ ] URL 使用英文 slug
- [ ] title、description、H1 唯一
- [ ] 正文 1,200–2,000 个汉字左右
- [ ] 更新时间对应实质更新
- [ ] 至少一个 H2，必要时使用 H3
- [ ] 有目录、FAQ、内链、外部权威来源
- [ ] 事实来源优先为官方或产品公开页面
- [ ] 价格、模型、政策注明核验日期
- [ ] 明确本站非 OpenAI 官方
- [ ] 第三方服务有账号、支付、隐私和稳定性提示
- [ ] 未引导绕过限制或规避风控

- [ ] ZEOGPT CTA 不超过 3 个且位置自然
- [ ] CTA 使用统一组件和 sponsored/nofollow
- [ ] 相关文章真实相关
