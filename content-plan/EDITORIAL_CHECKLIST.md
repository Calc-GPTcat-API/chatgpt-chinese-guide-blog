# Editorial Checklist

发布前全部勾选：

```text
[ ] 只解决一个明确问题
[ ] 与现有页面搜索意图不重复
[ ] 标题没有虚假“官方、无限、永久稳定”等承诺
[ ] 易变事实已用第一方来源核验
[ ] 至少有一个原创截图、实测、错误案例、表格或判断框架
[ ] 正文没有后台关键词串
[ ] 标签去重且不超过 4 个
[ ] 教程先回答问题，首屏没有裸推广条
[ ] 官方与第三方身份清楚
[ ] 推广链接已披露并使用 sponsored nofollow
[ ] 首次发布、最近更新、信息核验日期真实
[ ] 站内链接与外部来源可访问
[ ] contentStatus 与 published-articles.json 状态一致
[ ] npm run verify 通过
[ ] 桌面与手机预览通过
```
