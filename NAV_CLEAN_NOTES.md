# 导航清理版说明

本版按照“教程站优先，ZEOGPT 不抢主导航”的原则调整：

1. 顶部导航删除了“ZEOGPT 说明”。
2. `/zeogpt/` 页面没有删除，继续作为第三方服务说明页存在。
3. 顶部导航保留：
   - 首页
   - 教程博客
   - 套餐选择
   - 常见问题
   - 关于
   - 前往 ZEOGPT
4. ZEOGPT 页面仍可从文章、套餐页、FAQ、左侧 sidebar 和 CTA 进入。
5. 首页原格式不重做，只轻微弱化“ZEOGPT 广告站”的感觉。
6. npm 兼容处理保持：不携带 `node_modules` 和 `package-lock.json`，默认使用国内 npm 镜像。
