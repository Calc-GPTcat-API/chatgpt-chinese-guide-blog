# 本版修复说明

这版主要修 `/blog/` 页面：

1. `/blog/` 不再使用自定义大卡片首页组件，改成和文章详情页一样的 VitePress 文档布局。
2. 页面效果会接近文章页：左侧栏目导航，中间正文，右侧目录。
3. 保留原来的首页格式，不动首页绿色科技风、入口导航、按钮和卡片布局。
4. 保留字体优化，不再出现过粗、过大、英文小标题割裂的问题。
5. 保留 npm 兼容处理：不携带 `node_modules` 和 `package-lock.json`，默认走国内 npm 镜像。
