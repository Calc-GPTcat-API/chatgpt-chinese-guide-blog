# 站外信号执行清单

目标不是买垃圾外链，而是让核心页面有少量真实入口。

## 第一批推荐渠道

1. 知乎专栏：发文章摘要，链接回核心教程。
2. CSDN：发 ChatGPT 使用、OpenAI/AI 工具类技术教程。
3. B站动态/专栏：发“ChatGPT 国内使用避坑”摘要。
4. GitHub README：做一个 AI 工具导航或教程索引，放核心文章链接。
5. AI 工具导航站：提交首页、博客页和 3–5 篇核心教程。

## 执行原则

- 不买垃圾外链。
- 不用“官方中文版”“永久免费”“永久稳定”等词。
- 每个平台只发适合该平台的摘要，不要原文搬运。
- 优先前 20 篇核心文章。
- 每次更新文章后同步更新站外摘要。
