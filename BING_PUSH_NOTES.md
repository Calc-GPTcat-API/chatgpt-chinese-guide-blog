# Bing 冲刺版说明

这版按照参考站的优势做了增强，但保留我们自己的优点。

## 借鉴参考站的优点

1. 左侧导航更密，按搜索词分组。
2. `/blog/` 是文章索引页，不是营销卡片页。
3. 标题更像真实 Bing 搜索结果，覆盖“官网入口、国内使用、中文版、Plus 充值、镜像站安全吗”等搜索词。
4. 中间直接列出文章标题、日期、分类、摘要和标签。
5. 60 篇文章全部进入 sitemap 和 sidebar，方便抓取和内链。

## 保留我们的优点

1. 不把 ZEOGPT 和套餐选择放在顶部主导航。
2. 不写“官方中文版”“永久稳定”“百分百可用”“绝不封号”等高风险承诺。
3. 所有文章都有日期，避免参考站里出现的 Invalid Date 问题。
4. 保留首页原格式，不重做首页。
5. 保留 npm 兼容处理，不带 `node_modules` 和 `package-lock.json`。
