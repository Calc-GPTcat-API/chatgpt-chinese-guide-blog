# 简化导航版说明

本版按你的反馈处理：

1. 顶部导航改成：首页 / ChatGPT / 博客 / Gemini / 免责声明。
2. 删除顶部“常见问题”，FAQ 页面保留但不作为主入口。
3. 删除顶部显眼 ZEOGPT 入口，避免首页一眼像广告。
4. 左侧 sidebar 从多个细分类改成 5 个大组：
   - 使用指南
   - ChatGPT 教程
   - AI 模型
   - 多模型方案
   - 站点说明
5. ZEOGPT 不抢首页和顶部导航，但仍保留在：
   - 首页次级入口
   - 文章内 CTA
   - 文章底部 CTA
   - 多模型方案侧栏
   - 套餐页与 ZEOGPT 页

目标：像教程站接 Bing 流量，用户读到需要第三方方案时再自然进入 ZEOGPT。
